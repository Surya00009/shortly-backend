# 🚀 Shortly – Secure URL Management & Analytics Platform

A secure URL management platform built using Java, Spring Boot, Spring Security, JWT, Hibernate, MySQL, and React.

The application allows authenticated users to create, manage, and analyze shortened URLs through a secure dashboard while following a layered backend architecture.
